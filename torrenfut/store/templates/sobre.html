Sobre o Projeto

Este é um projeto de um e-commerce focado na venda de artigos esportivos, desenvolvido como parte da disciplina Implementação de Sistemas de Informação do curso de Sistemas de Informação na Universidade Estadual Paulista (UNESP) - Campus Bauru.

Tecnologias Utilizadas

  • Python: Linguagem de programação utilizada no desenvolvimento do backend.
  • Django: Framework para o backend, responsável por gerenciar o banco de dados, as rotas e a lógica de negócios.
  • HTML, CSS e JavaScript: Linguagens utilizadas no frontend, garantindo a estrutura, o design e a interatividade da aplicação.

Funcionalidades

O sistema oferece um conjunto completo de funcionalidades voltadas para a gestão eficiente de um e-commerce de artigos esportivos. Confira:

Gestão de Usuários

  • Cadastro de administradores com privilégios específicos.
  • Cadastro e gerenciamento de clientes e fornecedores.

Gestão de Produtos e Estoque

  • Cadastro, edição, exclusão e visualização de produtos (CRUD).
  • Cadastro de tributos, incluindo IOF e outros impostos aplicáveis.
  • Emissão de alertas automáticos para estoques baixos.
  • Emissão de relatórios detalhados de estoque.

Precificação

  • Cálculo automatizado do preço de venda baseado em:
    • Preço de custo.
    • IOF e outros tributos cadastrados.
    • Custos de frete.
    • Porcentagem de lucro configurável.

Gestão de Compras e Vendas

  • Recebimento e controle de pedidos de compra.
  • Realização e registro de vendas diretamente no sistema.
  • Emissão de confirmação de vendas com detalhes para o cliente.

Relatórios e Informações

  • Emissão de relatórios detalhados de vendas e estoque, permitindo análise do desempenho comercial.

Objetivo

O principal objetivo deste projeto é aplicar os conhecimentos adquiridos na disciplina, integrando os conceitos de desenvolvimento web com práticas modernas de programação. Além disso, visa proporcionar aos alunos a experiência de criar uma aplicação funcional e completa.

Equipe

O projeto foi desenvolvido por alunos do curso de Sistemas de Informação, sob a orientação do Prof. Dr. Higor Amario de Souza.
